# Twitter Follow Card (React)

Este proyecto es un ejercicio práctico desarrollado con React cuyo objetivo es entender y aplicar los conceptos fundamentales de la librería, especialmente:

-Componentes funcionales

-Hooks (useState)

-Reactividad

-Flujo de datos con props

-Renderizado condicional

-Manejo de eventos

La aplicación simula una tarjeta de seguimiento estilo Twitter, donde cada tarjeta representa un usuario y permite alternar entre los estados “Seguir” y “Siguiendo” de forma reactiva.

# Características actuales

-Componente reutilizable TwitterFollowCard

-Estado local para manejar el follow/unfollow

-Cambio dinámico de texto y estilos según el estado

-Uso de children para contenido flexible

-Estado levantado al componente padre (App) para manejar props dinámicas

-Avatares generados dinámicamente usando unavatar.io

# Objetivo del proyecto

Este proyecto forma parte de un proceso de aprendizaje y práctica con React, enfocado en:

-Comprender cómo funciona el estado

-Cómo React actualiza la UI automáticamente

-Separación de responsabilidades entre componentes padre e hijo
